How to Play

TNNT is hosted by hardfought.org.

Most people play TNNT with an SSH client to connect to the game servers. Linux and Mac OS typically come with this installed; Windows users can use either a free client like PuTTY or Windows Subsystem for Linux. SSH to "nethack@hardfought.org" or pick an alternate server based on whatever's closest to where you are geographically:

  • hardfought.org: USA (East Coast)
  • eu.hardfought.org: Europe (London)
  • au.hardfought.org: Australia (Sydney)

If you'd like to play with the TNNT tileset, or you're at a location that blocks everything except web access, you can also play in a Web browser using the hardfought web terminal.

If you do not already have an account on Hardfought, you need to register one first after connecting. Then, select the TNNT game option in the main menu after you've logged in. That's all there is to it!

To create, join, or manage clans, go here. Clan membership (creating and joining) will be frozen on November 10th at 00:00 UTC!

More Information

For more general information on NetHack, check out Wikipedia, the NetHackWiki, and The Greatest Game You Will Ever Play.

For more detailed information about the TNNT tournament, see the FAQ page.

TNNT chat discussion happens on the Libera.Chat IRC server, on the #tnnt and #hardfought channels, or via the Roguelike Discord on #nethack-tnnt (IRC to Discord bridge is active). All TNNT discussion is on-topic in these channels, and this is the best way to reach the admins if you have any issues. Click here to open IRC chat. Enter "#tnnt" for the channel.
